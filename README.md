## Installation
```
git clone https://github.com/gioacostax/app-web.git
cd app-web
npm install
```

## Development Mode
```
npm start
```

## Build Project
```
npm run build
```

## Features
- Hot Reloading (HMR)
- Styles preprocessor (Sass)
- Next Generation Javascript (Babel)
- Templating (Jade)
- Javascript Linter (Eslint)
